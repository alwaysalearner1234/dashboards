# 📊 Procedure Heart Attack Cost Analysis Dashboard

## 🚀 Overview
This project presents a **comprehensive data analysis dashboard** for analyzing the **cost of heart attack procedures** across different factors such as state, hospital ratings, and experience levels.

The dashboard provides **interactive visual insights** into cost distribution, helping identify trends and patterns in healthcare expenses.

---

## 📌 Key Insights

- 🏥 **Highest cost state:** California (~48L)  
- 📊 **Average procedure cost:** ~9.68L  
- 📉 **Readmission rating impact:**  
  - Max cost (Below): ~2C  
  - Min cost (None): 0K  
- ⚠️ **Mortality & Experience factors significantly affect cost trends**

---

## 📈 Dashboard Features

### 🔹 KPI Metrics
- Total procedure cost  
- Average cost by state  
- Rating-based cost comparisons  

### 🔹 Visualizations
- 📊 Cost distribution across ranges (0–6K, 12–18K, etc.)  
- 🍩 Experience-wise cost breakdown  
- 📉 Mortality & safety impact charts  
- 🗺️ State-wise cost mapping  
- 📌 Readmission vs cost ratio analysis  

---

## 🧠 Insights Derived

- Higher **experience ratings** often correlate with **higher procedure costs**  
- **Mortality and readmission ratings** influence pricing patterns  
- Significant variation exists across **different states**  
- Cost distribution is **skewed toward mid-high ranges**
